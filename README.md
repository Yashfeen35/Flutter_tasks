# Week 1: Basic Flutter Development and UI Building

## Learning Objectives
- Understand Flutter's basic structure.
- Build simple and responsive user interfaces.
- Learn navigation between screens.

---

## Tasks Completed

### 1. Environment Setup
- Installed Flutter SDK and set up Android Studio/VS Code.
- Created a new Flutter project named `flutter_login_navigation`.

---

### 2. Login UI
- Built a **login screen** with:
  - Two `TextFormField`s: one for **email** and one for **password**.
  - A `FlatButton` (or `TextButton`) for **Login**.
  - A `Text` widget for **"Forgot Password?"**.
- Used basic layout widgets like `Column`, `Row`, and `Container` to structure the UI.
- Implemented **form validation**:
  - Email field checks for proper email format.
  - Password field ensures it is not empty.

---

### 3. Navigation Between Screens
- Created a **Home Screen** as the second screen.
- Implemented navigation using: Navigator.push()
